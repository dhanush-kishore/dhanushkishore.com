---
title: Writing is thinking
date: 2026-07-10
---

I have been using ChatGPT since Oct 2022, around the time it was launched. Since then, I have extensively used LLMs to help me write, to the point that now in July 2026, I am struggling to write even a paragraph on my own without using an LLM. Even as I type out this post, I am struggling to put my words together - it is clear to me that my ability to write has been severely atrophied. 

This is the main reason for me to start writing and posting on my personal website - to bring back my ability to express my thoughts in writing, because as I have heard being said, writing is thinking. 

So I commit to writing every single post on this website without using an LLM, even to edit it. I will still use Claude Code / Codex for the code of the website though. 

My hope is that by posting regularly, even if not a single person every reads it, I am able to slowly bring my ability to write back. The website is intentional extremely minimal, simply because I am not really trying to attract any audience at all - its almost a semi private blog in that sense.
